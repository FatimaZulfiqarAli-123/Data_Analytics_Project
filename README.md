# Titanic EDA Project

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset.

## Skills Used
- Data Cleaning
- Data Visualization
- Matplotlib
- Seaborn
- Pandas

## Analysis Includes
- Missing value handling
- Histograms
- Bar charts
- Survival analysis
- Correlation heatmap

## How to Run
pip install -r requirements.txt
Open titanic_eda.ipynb
